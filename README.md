# Portfolio Website

## Overview

This is a simple personal portfolio website built with HTML, CSS, and JavaScript. It presents current projects, a short introduction about me, and my contact information in a clean and minimal layout.

## Features

* Project showcase with up-to-date work
* Short “About Me” section
* Contact information
* Responsive and lightweight design

## Technologies Used

* HTML5 for structure
* CSS3 for styling
* JavaScript for interactivity

## Purpose

The website serves as a central place to present my work, give a quick overview of who I am, and make it easy for others to get in touch.
